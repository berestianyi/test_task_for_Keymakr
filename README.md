# TEST TASK FOR KEYMAKR

## Getting started

- install [uv](https://docs.astral.sh/uv/getting-started/installation/)
- install [task runner](https://taskfile.dev/)
- Initialize the project

```bash
  task init
```

## TO RUN PROJECT

```bash
  task run-app
```
